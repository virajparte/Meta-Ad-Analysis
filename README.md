# Meta-Ad-Analysis
## Project Overview

This project delivers a **Meta Ad Performance Dashboard** to monitor and analyze the effectiveness of paid advertising campaigns running on Facebook and Instagram. The primary objective is to provide the marketing team with actionable insights into campaign reach, engagement, conversions, and budget allocation to optimize overall ad spend and strategy.

### Scope

The analysis is strictly limited to paid ad campaigns on **Facebook** and **Instagram**. It explicitly excludes Messenger, Audience Network, and all Organic engagement/metrics.

### Key Performance Indicators (KPIs)

The dashboard tracks performance across three main categories:

| Category             | Key Metrics Tracked                                                       |
| :------------------- | :------------------------------------------------------------------------ |
| **Reach/Engagement** | Impressions, Clicks, Shares, Comments, Total Engagements                  |
| **Efficiency**       | CTR (Click-Through Rate), Engagement Rate, Conversion Rate, Purchase Rate |
| **Cost/Budget**      | Total Budget, Average Budget per Campaign                                 |

### Required Visualizations

The dashboard includes the following critical views:

  * **Geographic Analysis:** Target Country (Map visualization)
  * **Demographics:** Target Gender (Donut Chart), Target Age Group (Bar Chart)
  * **Time-Based Trends:** Calendar Month (Calendar Heat Map), Weekly Trend (Stacked Column by Ad Type), Hourly Trend (Area Chart)
  * **Ad Type Breakdown:** Ad Type Performance Matrix

### Audience and Time Trends

  * **Demographics:** **Females (43%)** engage significantly more than Males (22%). Peak engagement occurs in the **20-30 age group**.
  * **Time of Day:** Engagement is highest in the **late afternoon & evening** window, specifically between **15:00 and 20:00 hours**.

### Ad Type Performance

  * **Best Performing:** **Video Ads** consistently deliver the highest CTR, Conversion Rate, and Engagement Rate.
  * **Strong Support:** Story Ads also perform well with high impression volume.
